# Widgets Layer - DecorAgent

## Purpose
Reusable UI blocks composed from shared and entities.

## Slices
- `product-card/` - Single product display card
- `product-grid/` - Responsive product grid
- `profile-hero/` - Profile header section
- `header/` - Site header navigation
- `footer/` - Site footer

## Rules
- Widgets are composition of shared UI + entities data
- Can import from: shared, entities, features
- Cannot import from: screens, app
